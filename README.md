# Little Scars

**Little Scars** is a 2D Unity narrative game that explores the emotional growth of adolescents. Players guide the protagonist through primary and side storylines, making choices that affect confidence, empathy, and social relationships.

---

## 🎮 Features

- **Interactive Narrative:** Chapter-based story spanning elementary to high school years.
- **Character Stats:** Track player attributes like Confidence, Empathy, and Loneliness.
- **Dialogue System:** Branching choices with multiple outcomes.
- **Mini-Games:** Embedded classroom quizzes and puzzles to engage players.
- **State Machine Movement:** Smooth character control with idle, run, jump, and ledge detection.

---

## 🛠 Installation

1. Clone or download this repository

Play the game at
Final Year Project 2.5D.exe
Press Play to run the game in the Unity Editor.

⚙ Usage
Use arrow keys or WASD to move the character.

Interact with NPCs to trigger dialogues and mini-games.

Choices affect player stats and story progression.

Pause menu allows saving/loading progress.
